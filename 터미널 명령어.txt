ls:해당 디렉터리에 있는 모든 파일 확인
	-l:파일의 크기등 상세 정보 확인
cd:해당 디렉터리로 이동
./:해당 파일 실행
jobs:현재 작업중인 프로그램 보여줌 
fg [num]:[num]번쨰 작업 실행
mv [디렉토리 1] [디렉토리 2]:디렉토리1을 디렉토리 2에 잘라내기
cp {디렉터리 1} [디렉터리 2]:디렉토리1을 디렉토리 2에 복사
rm [파일]:파일 삭제
==rm -rf [디렉터리]:디렉터리 삭제
exit:터미널 종료

git add [file]:git 업로드 1단계
git commit -a:git 업로드 2단계
git push:git 업로드 3단계
